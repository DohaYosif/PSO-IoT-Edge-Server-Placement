# Optimized Localization of Edge and Server Nodes in IoT Networks

This project uses Particle Swarm Optimization (PSO) to determine the optimal placement of Edge and Server nodes in an IoT network. The optimized topology is evaluated in Contiki-NG using the RPL routing protocol.

## Technologies
- PSO using MATLAB
- RPL & IoT Simulation using Contiki-NG_cooja

## Performance Metrics
- Latency
- ETX
- Load
- Energy Consumption

## Results
The proposed PSO-based approach improves latency, energy efficiency, load balancing, and overall network performance.
